# Simple Code Editor

A lightweight desktop code editor built with Python and Tkinter.

This project is a simple GUI-based code editor that supports multiple tabs, file opening/saving, and running scripts directly from the editor. It was built to practise Python GUI development, file handling, subprocess execution, and basic IDE-style functionality.

## Features

- Multi-tab code editor
- Open existing files
- Save and Save As support
- Run Python, Lua, and JavaScript files
- Auto language detection based on file extension
- Manual run mode selection
- Output panel for stdout and stderr
- Dark UI theme
- Line numbers
- Fullscreen mode
- Keyboard shortcuts
- Stop running process button
- Clear output button

## Requirements

- Python 3.10+
- Tkinter

Tkinter usually comes included with standard Python installations.

Optional requirements:

- Node.js for running JavaScript files
- Lua or LuaJIT for running Lua files

## Supported Languages

| Language | File Extension | Requirement |
|---|---|---|
| Python | `.py` | Python installed |
| JavaScript | `.js` | Node.js installed |
| Lua | `.lua` | Lua or LuaJIT installed |

## How to Run

Clone the repository or download the project files, then run:

```bash
python simple_code_editor.py
```
